# 👕 Clothing Store Website

A modern and visually appealing static clothing store website built using **HTML5** and **CSS3**. The project focuses on clean UI design, structured layout, and visually engaging presentation.

## ✨ Features

* 🏷️ Well-structured layout using semantic HTML5 elements
* 🎨 Clean and modern UI design
* 🧩 Organized sections (Header, Products, Footer)
* 🛍️ Product cards with images and descriptions
* 📐 Consistent spacing, typography, and colors
* 🖥️ Designed primarily for desktop view

---

## 🛠️ Tech Stack

* 🌐 HTML5
* 🎨 CSS3
* 📦 Flexbox & CSS Grid

---

## ⚙️ Installation

1. Clone the repository:

```bash
git clone https://github.com/Hazem-2002/Static-Clothing-Store.git
```

2. Navigate to the project folder:

```bash
cd Static-Clothing-Store
```

3. Open the project:

Open `index.html` in your browser
